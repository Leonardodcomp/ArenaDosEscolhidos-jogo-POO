# ArenaDosEscolhidos-jogo-POO
Jogo criado para compor parte da nota da disciplina de Programação Orientada a Objetos


Observações iniciais:
1- Nossos personagens possuem nome de batismo, então já vem com nome pré-definido

2- Para integrar um RPG usual, estamos usando a biblioteca java.util.Random para usar o método random.nextInt(), esse terá a função de dado, ele ajudará também nos calculos de probabilidade
